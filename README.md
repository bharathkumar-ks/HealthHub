## Overview
Clinic Management System is a JSP/Servlet-based web application designed to manage clinic operations such as patient records, appointments, and administrator activities. The system allows administrators to add and manage clinic data efficiently through a web interface.

## Features
- Add and manage patient records
- Appointment scheduling and tracking
- Administrator module
- Input validation and error handling
- DAO-based database operations
- Layered architecture (Bean, DAO, Service, Servlet)

## Tech Stack
- Java
- JSP & Servlets
- JDBC
- HTML/CSS
- Apache Tomcat Server
